# Figma 仪表板设计规范

## 1. 总览
- 适用场景：文化遗产数字化学习进度、技能成长、任务提醒
- 设计风格：现代、扁平、色彩分明，支持暗/亮模式
- 主要模块：学习进度看板、技能树、时间轴、成就徽章、热力图

---

## 2. 组件规范

### 2.1 学习进度看板
- 组件类型：进度条、饼图、柱状图
- 数据结构：
```json
{
  "total_hours": 32,
  "course_hours": {"PyTorch深度学习": 12, "3D建模": 8, "前端开发": 6},
  "completed_tasks": 18,
  "planned_tasks": 24
}
```
- 交互：点击课程显示详细任务列表
- 颜色建议：
  - PyTorch深度学习：#FF6F61
  - 3D建模：#4ECDC4
  - 前端开发：#556270

### 2.2 技能树
- 组件类型：树状图/分层圆环
- 数据结构：
```json
{
  "AI算法": 80,
  "3D建模": 60,
  "前端开发": 40,
  "后端开发": 30,
  "区块链": 10
}
```
- 交互：点击节点展开子技能进度
- 颜色建议：每个领域独立色，主色+浅色渐变

### 2.3 时间轴
- 组件类型：水平时间轴、关键节点卡片
- 数据结构：
```json
[
  {"date": "2026-01-15", "event": "期末考试", "type": "考试", "priority": "P0-极高"},
  {"date": "2026-03-01", "event": "项目中期答辩", "type": "答辩", "priority": "P1-高"}
]
```
- 交互：鼠标悬停显示详细说明

### 2.4 成就徽章
- 组件类型：圆形徽章、进度环
- 数据结构：
```json
[
  {"name": "连续学习30天", "icon": "🔥", "achieved": true},
  {"name": "完成全部AI课程", "icon": "🤖", "achieved": false}
]
```
- 交互：点击徽章显示获得条件

### 2.5 热力图
- 组件类型：日历热力图
- 数据结构：
```json
{
  "2025-11-20": 2,
  "2025-11-21": 1.5,
  "2025-11-22": 2.5
}
```
- 交互：点击日期显示学习内容
- 颜色建议：低时长浅色，高时长深色（如 #E0F7FA → #00796B）

---

## 3. 设计尺寸建议
- 主看板宽度：1200px，高度：800px
- 进度条高度：24px，技能树节点直径：48px
- 时间轴节点间距：80px
- 徽章直径：64px
- 热力图单元格：24px x 24px

---

## 4. 交互与动画
- 进度条/饼图：动态增长动画
- 技能树节点：展开/收缩过渡
- 时间轴节点：弹出卡片动画
- 徽章：获得时闪烁/放大
- 热力图：hover 高亮

## 6. 设计工具建议
- Figma 组件库：建议分模块建立，便于复用
- 色板：主色+辅助色+状态色，支持暗/亮模式切换
- 字体：推荐 Inter、PingFang SC、Roboto
- 图标：可用 Figma 社区免费图标库

---

## 7. 交付说明
- 设计师可直接复制 JSON 示例作为数据源
- 前端开发可按数据结构对接 API 或本地 mock
- 交互动画建议用 Figma Prototype 演示

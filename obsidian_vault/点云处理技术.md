# 点云处理技术

## 什么是点云

点云是三维空间中点的集合，每个点包含 (x, y, z) 坐标信息，可能还包含颜色 (RGB) 和法向量等属性。

## 主要处理步骤

### 1. 点云获取
- 激光扫描仪
- 深度相机 (RealSense, Kinect)
- [[三维重建基础]]中的 MVS 算法

### 2. 预处理
- **降采样**: 减少点云数量，提高处理效率
- **滤波**: 去除噪声点
  - 统计滤波
  - 半径滤波
- **法向量估计**: 计算每个点的表面法向量

### 3. 配准
- ICP (Iterative Closest Point) 算法
- 多视角点云融合
- 全局优化

### 4. 表面重建
- Poisson 重建
- Ball Pivoting 算法
- Marching Cubes

## 常用工具

- **PCL (Point Cloud Library)**: C++ 点云处理库
- **Open3D**: Python 友好的 3D 数据处理库
- **CloudCompare**: 可视化和处理工具

## 文化遗产应用案例

使用点云技术可以：
1. 高精度数字化古建筑
2. 变形监测
3. 虚拟展览
